```
Which of the following are not legal constants in C? Classify each legal constant as either
integer or floating-point.
(a) 010E2
(b) 32.1E+5
(c) 0790
(d) 100_000
(e) 3.978e-2
```

**Solutions**  
(a) Floating-point
(b) Floating-point
(c) Not legal
- Explanation:  
    Octal constants contain only digits between 0 and 7  

(d) Not legal
- Explanation:  
    underscore not allowed

(e) Floating-point
