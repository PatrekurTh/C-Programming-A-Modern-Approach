```
Assume that a program contains the following declarations:
char c = '\1';
short s = 2;
int i = -3;
long m = 5;
float f = 6.5f;
double d = 7.5;
Give the value and the type of each expression listed below.
(a) c * i 
(b) s + m 
(c) f / c 
(d) d / s 
(e) f - d
(f) (int) f
```

**Solutions**  
(a) Value:   -3 Type: int  
(b) Value:    7 Type: long  
(c) Value:  6.5 Type: float  
(d) Value: 3.75 Type: double  
(e) Value: -1.0 Type: double  
(f) Value:    6 Type: int
